# Computational-Thinking-Learning-Outcome-Analysis

## Project Overview
This project analyzes the effectiveness of computational thinking training conducted in government schools for students of grades 6, 7, and 8.

The analysis compares Baseline and Endline assessment scores to measure student improvement after the training program.

## Tools Used
- Tableau
- Microsoft Excel

## Key KPIs
- Total Students
- Participating Schools
- Average Baseline Score
- Average Endline Score
- Improvement Percentage
- Skill-wise Performance Analysis

## Key Insights
- Overall student performance improved by 27.6%
- Tool-Based learning showed highest improvement
- Grade-wise and gender-wise comparisons were analyzed
- School-level performance gaps were identified

## Dashboard Features
- Interactive filters
- Baseline vs Endline comparison
- Skill-wise analysis
- School-wise performance tracking
- Gender and grade segmentation
